Gulp-Webpack сборка

Для работы сборки у вас должны быть установлены глобально:

1) npm 5 (для Windows 10, версия должна быть не выше 5.3  npm install npm@5.3.0 -g )
2) gulp-cli

Как установить проект:

1) clone this repo
2) npm i
3) gulp4 (npm i github:gulpjs/gulp#4.0)
